# Spkr

Aplicación de aprendizaje de idiomas inspirada en Duolingo.

## Descripción

Spkr es una aplicación educativa que permite aprender idiomas mediante ejercicios interactivos y flashcards, con diferentes estrategias de aprendizaje.

## Tecnologías

- Java
- Maven
- JPA (persistencia)
